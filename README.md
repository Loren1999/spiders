 - random 
 - lxml 
 - bs4 
 - fake_useragent

# 🌚问题
lxml中没有etree模块的问题解决方法：

- 去下面的地址下载对应自己python版本的lxml插件
- https://www.lfd.uci.edu/~gohlke/pythonlibs/#lxml 
- 下载后使用pip install 下载的文件名进行安装 
- 安装后重启PyCharm
